Legend of Mushroom Guia de aula de arqueiro

Conteúdo

Introduction to Legend of Mushroom Guia de aula de arqueiro

Para começar o Legend of Mushroom Guia da aula de Archer, gostaria de explicar que em diferentes partes do Legend of Mushroom para o qual você precisará de diferentes habilidades/amigos para se mostrar eficaz a fim de seguir em frente. Por exemplo, nosso guia de principais habilidades/amigos pode não funcionar para você, mesmo que você tenha todas as habilidades e amigos para usá-lo. Cada parte do jogo requer um conjunto necessário de habilidades/amigos para conquistá-la. Use o conhecimento que você aprende aqui como base para melhorar, adquirir conhecimento e progredir ainda mais.

Lembrete: Arqueiros são fortes contra Magos e fracos contra Guerreiros.

PVE completo Legend of Mushroom Guia de aula de arqueiro

Guia de classe de arqueiro PVE em estágio inicial

Amigos/habilidades úteis no estágio inicial


Primeiro, abordaremos os amigos/habilidades mais úteis, na minha opinião. Isso ajudará tanto no início quanto no final do jogo:

  • Arvoredo de Espinhos (PVE)
  • Surto de Velocidade (PVE/PVP)
  • Hero Bird (Tem múltiplas raridades, não apenas vermelho) (PVE/PVP)
  • Hydrosprite (PVE às vezes PVP)
  • Caracol (PVE)

Vou explicar por que sugiro estas habilidades/amigos:
Então, se você estiver usando caracol e arbusto de espinhos, isso atrasará os oponentes para chegar à posição de combate antes que eles possam começar a atacar você, Hero Bird aumenta sua velocidade de ataque e o mesmo com o aumento de velocidade, ambos se acumulam aumentando sua velocidade de ataque muito mais para acertar mais combos / críticos por segundo, aumentando seu DPS.

Hydrosprite é um bom amigo de se ter, se você estiver enfrentando um chefe e morrer antes de conseguir um segundo conjunto de habilidades, mesmo com Hydrosprite equipado, você pode removê-lo e alterá-lo para outra coisa.

Dragões bons para PVP e PVE:

Os dragões são bons de usar, pois aumentam muito o seu Crit DMG. No início do jogo, use o Fortune Dragon, no final do jogo, use o Treasure Dragon, pois o Treasure Dragon é a variação mais alta dele.

Agora que examinamos uma variedade de diferentes habilidades/amigos que são úteis, vamos ver o que todos vocês vieram fazer aqui e quais são as configurações recomendadas de amigos/habilidades/relíquias. Portanto, farei o meu melhor para ajudar a fornecer isso e também para orientá-lo em meu raciocínio. Começarei primeiro com PVE e depois farei PVP.

Vamos começar com o início do jogo e avançar no jogo.

Eu recomendaria este carregamento a partir de quando você puder adquirir as habilidades/amigos para usá-lo. Você pode usar este carregamento até adquirir a variação maior dele.

Habilidades iniciais para Legend of Mushroom Classe de Arqueiro

  • Thorn Thicket, isso é bom para atrasar o oponente. Essa habilidade funciona bem com caracol.
  • Speed ​​​​Surge, esta é uma boa habilidade para aumentar sua velocidade de ataque, funciona bem com um amigo buff de velocidade de ataque.
  • Shroom Shield, esta é uma boa habilidade para usar como proteção extra até desbloquear Clone Strike. O escudo Shroom funciona bem com a Estátua de Energia.
  • Renovação da Natureza, isso ajudará principalmente em estágios, pois irá regenerar 30% do seu HP máximo, e também funciona bem com a Estátua de Energia para causar dano adicional. Se esta habilidade não estiver funcionando bem para os Chefes, você pode considerar alterá-la para Coin Bomb*
  • Coin Bomb é bom para causar o dano adicional necessário para eliminar o chefe.
  • Vá devagar, esta é uma boa habilidade contra mobs de palco e também contra chefes, pois reduz a velocidade de ataque do oponente. Se você já tem debuffs por sua lentidão antes que eles cheguem até você, então esta é uma boa habilidade para debuffá-los novamente.

Amigos em estágio inicial para Legend of Mushroom Classe de Arqueiro

  • Caracol, este é um bom amigo para desacelerar o oponente. Este amigo não funciona para PVP.
  • Banana, essa banana é uma ótima amiga, pois aumenta o dano do combo e a% do combo, isso vai ajudar principalmente se você usar a Crit Mask Relic.
  • Hydrosprite, este amigo só é bom se você tiver tempo suficiente em sua batalha para usar suas habilidades inúmeras vezes. Se você não tiver tempo suficiente para usar a mesma habilidade duas vezes, não use esta e use um amigo alternativo. Uma alternativa para isso poderia ser um amigo de habilidade ou um amigo de dano de ataque básico. Sinta-se à vontade para trocar entre eles e ver o que funciona para você (eu recomendo o amigo de dano de ataque básico para dano de habilidade no início do jogo)
  • Rainshade Bird, esse pássaro tem variações maiores dele, esse amigo aumenta sua velocidade de ataque o que ajuda muito para um arqueiro, maior velocidade de ataque significa acertar mais combos, seus combos também podem criticar então isso aumenta suas chances de acertar mais críticos.
  • Fortune Dragon, o dragão da fortuna é um ótimo amigo, pois adiciona um aumento global de 20% no dano crítico ao seu dano crítico atual. % global significa apenas que adicionará 20% a esse valor total. Se você tiver um valor total de 1000% de dano crítico, isso totalizará 1200%, o que soma e ajuda muito.

Relíquias de estágio inicial para Legend of Mushroom Classe de Arqueiro

Agora, vamos discutir algumas relíquias para usar neste carregamento. Eu recomendaria as seguintes relíquias:

A Máscara colada é bom para a taxa crítica adicional.

A Pipa Invocadora do Trovão é bom para aumentar o valor do dano à medida que aumenta de nível. Alternativamente, você pode usar o Dragon Weave se este não for aumentado de nível.

A caixa Mágica. Se você adquiriu esta relíquia, você pode trocar o Hydrosprite pelo Deer, caso ele não esteja se mostrando tão eficaz em lutas contra chefes.

A Estátua de Energia é bom para quando você está usando Natures Healing/Shroom Shield, mais eficaz quando usa ambos.

Livro de Chamas – Este livro é bom para ir contra fases, em alguns casos contra alguns chefes (Ex. Chefes que precisam estar próximos a você para atacar. Chefes como o dragão que atacam quase quando aparecem podem não ser úteis, se este for o caso, então o livro de imunidade será mais útil.)

Besta Esconder Livro – Este livro é bom contra chefes para causar dano extra por segundo ao chefe. Se o chefe tiver que viajar uma longa distância antes de atacar você, então este livro é ideal, se o chefe for como o dragão e percorrer apenas uma curta distância antes de entrar sua postura de ataque, então o Livro de Imunidade será mais útil.

Livro de Imunidade – Este livro é bom com chefes que atacam quando aparecem, isso lhe dará alguns segundos de imunidade para que não prejudique tanto sua saúde, ou seja, você viverá por mais tempo.

Guia de classe PVE Archer de estágio intermediário

Agora vamos entrar em uma construção do tipo meio do jogo para isso. Fornecerei várias versões que você pode usar se adquirir uma habilidade/amigo rosa.

Habilidades de estágio intermediário para Legend of Mushroom Classe de Arqueiro

  • Bomba de fumaça – Esta habilidade é boa para causar dano adicional, uma boa relíquia para usar nesta habilidade é a Estátua do Tempo para aumentar a duração da habilidade em 50%. Permitindo que você cause 30% de dano extra não por 5 segundos, mas por 7.5 segundos.
  • Desarmar – Esta habilidade desarma oponentes por 3 segundos se você estiver usando estátua do tempo, então serão 4.5 segundos que os mobs/chefes do palco serão desarmados, se você estiver usando arbusto de espinhos + caracol então você pode atrasar seu desarmamento para cerca de 6 segundos dependendo do chefe/velocidade dos mobs do palco. Você deseja principalmente desarmar para o ponto mais próximo de onde os mobs começam a atacar você. Para testar isso, ative um atraso de 5 segundos se você for atacado antes de ativar e depois diminua o segundo. Este atraso irá variar entre diferentes fases/chefes.
  • Surto de velocidade – Esta habilidade é boa especialmente ao usar o herói pássaro amigo, já que a velocidade de ataque de ambos se acumula, aumentando ainda mais sua velocidade de ataque.
  • Arvoredo de Espinhos – Essa habilidade é boa para atrasar os mobs do palco, pode ajudar também nas batalhas contra chefes dependendo do alcance de ataque do chefe. Se for um dragão, por exemplo, que ataca a longa distância, então os espinhos podem não ser muito úteis para você; se for esse o caso, você pode trocá-lo por uma bomba de moeda contra chefes.
  • Escudo de Cogumelo – Isso ajudará a fornecer saúde adicional para dominar o chefe tanto na Defesa quanto no Ataque.
  • Rajada Selvagem – Isso ajudará nos estágios em que você pode alternar entre escudo de cogumelo ou rajada selvagem, sugiro rajada selvagem, pois causa uma quantidade razoável de dano AOE e dá a você a opção de usar a Foice Carmesim

Habilidades adicionais que você pode usar se as tiver desbloqueadas:

  • Ataque Blitz – Se você tem essa habilidade e não as outras, então você pode usar o colar Spirit, para esta habilidade você deseja que ele seja ativado depois que seu desarmamento acabar, antes de você começar a ser atacado novamente. Tenha em mente que o desarme estará ativo por 4.5 segundos, se estiver com um atraso de 4 segundos, então você desejará que seu atraso seja 4+4.5=8.5, no entanto, não há atraso de 5 para esse intervalo, então diremos 8 segundos. Não estou dizendo que este é o momento que você deve colocar apenas um exemplo de como utilizar os atrasos.
  • Golpe de clone – Se você tem essa habilidade, ótimo. Esta habilidade faz maravilhas para PVE e PVP, especialmente no meio do jogo. Clone só é inútil se ele morrer instantaneamente em PVP antes que seu desarme / blitz expire. Se você estiver usando esta habilidade para PVE/Chefes, use o colar Storm para aumentar o dano do clone.
  • Armadilha Mundana – Esta habilidade pode ser boa para estágios se você tiver uma taxa crítica baixa, isso pode ajudar a aumentar essa taxa crítica e ajudá-lo a causar dano adicional, nem sempre será útil, mas se você estiver faltando as outras habilidades rosa, então esta é uma opção adicional.

Amigos intermediários para Legend of Mushroom Classe de Arqueiro

  • Caracol – O caracol é uma boa escolha principalmente em fases, mas tem dificuldades em algumas batalhas contra chefes. Se o chefe tiver um ataque alto, como o dragão, então é melhor não usá-lo e substituí-lo por outra coisa, como o cervo.
  • Hidrosprite – Este é um bom amigo porque recarrega suas habilidades mais rapidamente, proporcionando mais buffs de velocidade de ataque devido ao aumento de velocidade e mais armadilhas de Thorn Thicket. Se você está enfrentando um chefe e morre antes de poder ativar suas habilidades novamente, então você pode usar um amigo alternativo, uma boa alternativa para isso poderia ser #1 Benny, o coelho #2 Berinjela com palito #3 Capuz de frango
  • Herói Pássaro – Este amigo é uma boa opção, pois aumentará sua velocidade de ataque e acumulará com a habilidade de aumento de velocidade.
  • Banana – A banana é boa, se você não tiver a banana vermelha desbloqueada e apenas a azul então tudo bem, isso ainda funcionará no início/meio do jogo, no final do jogo a banana vermelha é mais preferível.
  • dragão – Se você tem o dragão do tesouro, então essa é uma alternativa melhor ao dragão da fortuna.
  • Veado – Este amigo é bom para resistência a danos e se você estiver usando este amigo, você pode usar a relíquia para isso. A relíquia fornece dano adicional para o próprio amigo cervo causar aos oponentes.

Usar o caracol depende do chefe que você está enfrentando. Se o chefe for um chefe com um grande alcance de ataque e começar a atacar ao ser gerado, você poderá substituir o caracol por um cervo anjo e equipar a relíquia da caixa mágica.

Guia de classe PVE Archer de estágio final

Agora que discutimos o início/meio do jogo para Legend of Mushroom Classe Archer vamos entrar em uma seção mais avançada do jogo, esses conjuntos são ideais para o final do infinito – revelação, tenho usado isso através da revelação. Os testes ainda não foram feitos para o milagre (o estágio após a revelação 10-10), pois isso está mais adiante no jogo

Legend of Mushroom Construção de dano total do arqueiro

Primeiro discutiremos o dano puro com base na maior quantidade de dano causado. Isso será ideal para chefes como Ladrão de lâmpadas, Showdown Boss ou, em alguns casos, Blazing Cave.

  • Se você não tem o amigo Polvo, então use Berinjela
    • Se não houver berinjela, use Fiery Tale.

Legend of Mushroom Construção de defesa total do Archer

Construção defensiva completa, sugerida principalmente apenas para Grumpy Big Fish (Family Fish Boss localizado na área da família).

  • Se você não tem o amigo Puppy Fevor, use o Panda.
  • Se você não tiver B. Duck, use Hero Bird.

Legend of Mushroom Construção do estágio do arqueiro

Este próximo carregamento será uma boa base para ajudá-lo a eliminar mobs de palco.

  • Desarmar: Tente começar sem atraso, se nenhum atraso não ajudar, então você pode tentar adicionar cerca de 4 segundos de atraso, isso varia, se você chegar ao estágio 3 de 4, por exemplo, e seu Desarmar ainda estiver em espera, você pode remova o atraso e tente novamente,

Legend of Mushroom Construção de chefes de estágio de arqueiro


Carregamento de Stage Bosses, também pode ser usado para outros chefes, como testes sombrios, testes de alma ou caverna em chamas, em alguns casos isso também pode funcionar para teste de ascensão na nuvem (varia de acordo com os chefes, se não tiver um chefe e apenas mobs de palco substituem o livro de couro de fera pelo livro de chamas).


Se você estiver enfrentando um chefe como um dragão que tem um grande alcance de ataque e ataca quase instantaneamente no spawn, então o conjunto acima parecerá menos eficaz devido ao caracol e ao arbusto de espinhos não desacelerar o oponente tanto quanto aquele que faria chegar até você com um curto alcance de ataque, se for esse o caso, você pode tentar este conjunto abaixo.

  • Atrasos: Atraso deslumbrante de 1 a 2 segundos, você deseja cronometrar antes que o chefe inicie o ataque.

Dazzled será ativado por 1.5 segundos (1 segundo e meio) e tem uma duração estendida de 50% a partir da estátua do tempo, o que significa que 50% de 1.5 será 0.75. O que é quase um segundo inteiro. Portanto, 1.5 se transformará em aproximadamente 2 segundos. Você gostaria de ativar seu desarme para quando o atordoamento terminar e os oponentes estiverem prestes a iniciar um ataque.

Digamos que você ficou deslumbrado com o atraso e será iniciado 2 segundos na batalha, o que significa que ele ficará ativo por aproximadamente 2 segundos, o que significa que deve terminar aproximadamente 4 segundos após o início da batalha. O oponente precisará de um pouco de tempo para se recuperar da habilidade e entrar em posição de combate e com o cálculo de deslumbrado sendo uma estimativa você ainda tem algum tempo sobrando, então você pode usar 4.5 segundos para o desarme ser ativado, cada chefe será diferente então não deixe de brincar com isso.

Você também pode tentar mudar o deslumbramento para o ataque blitz, se você decidir fazer isso, então você vai querer usar o desarme para quando o oponente estiver prestes a atacar, então aproximadamente 4 segundos ou mesmo nenhum atraso no desarme durará 3 segundos + 50% ( 1.5 segundos), o que significa que terminará em 4.5 segundos. Se você não tiver atraso para desarmar, poderá ativar seu ataque blitz em 4.5 segundos/4 segundos.

Guia de classe de arqueiro PVP em estágio inicial

Amigos / habilidades úteis para PVP


Agora vamos discutir o PVP e o que é melhor usar em diferentes pontos do jogo, lembre-se de que diferentes oponentes podem precisar de diferentes habilidades/amigos para derrotá-los. Essas habilidades/amigos são provavelmente as mais importantes para o PVP.

Em primeiro lugar estão as habilidades:

Habilidade Extra: Clone Strike – use apenas se seu clone não morrer antes que seu desarme/blitz expire. Se ele morrer antes de causar dano ao oponente, remova-o.

Os seguintes amigos são principalmente de final de jogo, mas o dragão tem uma versão laranja, a banana tem uma versão azul e o Hero Bird tem várias versões inferiores.


Primeiro discutiremos algumas habilidades/amigos que você pode usar em diversas partes do jogo para ajudar a otimizar suas construções.


Shroom Shield, essa habilidade é boa para pegar o escudo, é boa para usar até conseguir Clone Strike / Blitz Assault.


Speed ​​​​Surge, esta é uma habilidade inicial do jogo e funciona bem para arqueiros em todos os aspectos do jogo, usar isso e um amigo de velocidade de ataque aumentará sua velocidade de ataque muito mais para que você acerte mais combos/crits (lembre-se de que seus combos podem criticar)


Vá devagar, esta é uma habilidade do jogo, você provavelmente a obterá no primeiro dia em que o jogo for lançado ou em alguns dias com base em sua progressão. Esta habilidade é boa para usar em PvP se você estiver enfrentando outro Arqueiro ou até mesmo um Guerreiro para reduzir sua velocidade de ataque.


Easy Breezy, pode ser usado contra Magos para reduzir o dano de ataque em vez de reduzir a velocidade de ataque usando a velocidade lenta. Os magos não têm uma velocidade de ataque alta, então é mais ideal para usar contra magos.


Smoke Bomb, esta habilidade aumenta o dano que você causa aos oponentes em 30% por 5 segundos (7.5 segundos se você usar a estátua do tempo)


Blitz Assault, esta habilidade permite que você fique invencível por 3 segundos, dando-lhe uma vantagem extra em saúde, tenha em mente que existem skins avançadas que ignoram os efeitos do status de imunidade. Mas essa habilidade é sempre boa de usar, especialmente no meio do jogo


Desarmar, esta habilidade é vital para PvP, especialmente com a estátua do tempo para obter tempo adicional para cancelar o ataque do oponente, ajudará muito contra guerreiros/arqueiros, se for contra magos, a habilidade deslumbrada será mais eficaz. (Isso pode não impedi-los de lançar algumas habilidades, mas pelo menos atrasará a taxa de recarga de suas habilidades, o que ajuda os magos a terem uma taxa de atualização rápida em comparação com outras classes).


Hydrosprite, isso só é bom se suas batalhas durarem muito tempo, se suas batalhas durarem menos de 10 segundos não adianta usar esse pal, são múltiplas variações: Vermelho, Laranja, Amarelo, Roxo e Verde.


Dragão do Tesouro, esse amigo é muito bom, pois aumenta globalmente o seu dano crítico, o que aumenta muito o seu dano crítico global. Este amigo tem 2 variações: Rosa e laranja.


Banana, este amigo é bom porque aumenta a% da sua taxa de combo, bem como aumenta o dano do combo. Existem 2 variações: Vermelho e Azul. O vermelho é obtido em um evento de gasto. (O evento ocorre aproximadamente uma vez por mês)


Hero Bird, este é um pássaro de velocidade de ataque e aumenta sua velocidade de ataque para que você cause mais combos/críticos, causando mais dano ao oponente. Existem múltiplas variações para este pássaro: Vermelho, Laranja, Amarelo, Azul e Cinza

Habilidades iniciais para Legend of Mushroom Classe de Arqueiro

Nota: Se estiver usando Meteor Fall, você pode usar um amigo para aumentar o DMG de habilidade e também para aumentar a produção de DMG de habilidade (isso é bom apenas no início do jogo). Alternativamente, use a bomba de moeda.

Relíquias de estágio inicial para Legend of Mushroom Classe de Arqueiro

Você pode alternar entre o livro de tomo de pedra ou o livro de chamas. Eu pessoalmente prefiro o livro de tomo de pedra no início do jogo e usar o livro de chama mais tarde no jogo. Eu inseri Skill DMG na equação porque no início do jogo você não terá estatísticas perfeitas, você apenas estará tentando correr para obter equipamentos melhores para obter o aumento de ATK, HP e DEF e não teria se concentrado muito em seu Estatísticas. Estátua de Oração para Legend of Mushroom Classe de Arqueiro

Vamos discutir outros aspectos do jogo. Estaremos agora discutindo Estátua de Oração e o que é melhor e por quê?

Idealmente, em sua estátua de oração, você deseja 5 afixos combinados como abaixo:

Testei diferentes carregamentos, por exemplo;

  • 4 Combos + 1 Dano de Chefe
  • 4 Combos + 1 Dano Crítico

O combo dmg esmagador que a estátua dá é obrigatório e vai ajudar muito na sua progressão, use isso tanto para pvp quanto para pve.

NOTA: Algumas pessoas têm dificuldade em entender as porcentagens, então aqui explicarei rapidamente os buffs.

Usaremos este como exemplo.

Esse Combo Global DMG + 72.60%, Mas o que isso representa?

Esses 72.6% são um aumento global (observe que o valor da estátua é apenas tão grande porque leva todos os 5 em consideração)

Se você clicar no perfil de um jogador para comparar estatísticas, localize o combo DMG e clique na guia de informações para ver de onde vem o dano do combo. O aumento global é o maior aumento que você pode obter em suas estatísticas, então isso é multiplicar seu outro dano de combo por esse valor.

CONCLUSÃO: O ganho de 72.6% é adicionado às suas estatísticas e é o aumento do próprio dano do seu combo.

Agora vamos discutir os 91%. O que é isso?

Para explicar isso, mencionarei novamente que 91% representa 72.6%, quanto maior o valor de 91%, mais rara é a estatística e representa o quão melhor você poderia obter.

Portanto, 91% dá 72.6% de Combo DMG NO ENTANTO. Se olharmos para os 99%

Você pode ver que dá um grande combo% de dano, isso mostra que 99% é um dos melhores resultados que você pode obter (100% será o mais alto)

O valor% do dano do combo global aumentará conforme a raridade aumenta, por exemplo, 100% no SSS provavelmente atingirá o máximo de 80% de aumento de dano de combo, enquanto SSS + provavelmente atingirá o máximo de 100% de aumento de dano de combo.

Almas para Legend of Mushroom Classe de Arqueiro

Primeiro, vamos começar com a ordem de prioridade para desbloquear almas cor de rosa, vou listá-las em ordem decrescente (ou seja, de cima, primeiro, de baixo, por último)

  • Combo DMG + Boss DMG
  • ATK Global + Taxa de Cura
  • Bônus global de HP + HP Regen
  • Defesa + resistência ao DMG do chefe
  • ATK básico + DMG de habilidade
  • Res Crítico + Pal Crítico
  • Dano Crítico + Dano Crítico de Habilidade
  • Contador + Pal DMG

    Explicação:

Combo DMG + Boss DMG

  • Pink Combo DMG vs Red não aumenta o valor geral do Combos, a única vantagem adicional que você obterá é o Boss DMG, no entanto, isso é muito útil para inúmeras coisas, como Soul Trial, Dark Trial, Lamp Thief, Server Showdown Boss, etc. Basicamente todos os chefes)

ATK Global + Taxa de Cura

  • Ataque Global Rosa vs Ataque Global Vermelho. Pink tem um valor de ataque global mais alto e aumentará sua produção de dano. A taxa de cura é boa, pois ocorre em todos os ataques.

Bônus global de HP + HP Regen

  • HP rosa vs HP vermelho. Pink HP tem um valor base mais alto do que a alma vermelha, você também ganhará HP Regen, que irá curar uma% do seu HP a cada segundo que a batalha continuar.

Defesa + Boss DMG Res

  • Pink Defense vs Red Defense, Pink Defense tem um valor mais alto do que Red Defense, você também ganhará uma Boss DMG Resistance que é boa para ajudá-lo a aumentar sua posição contra chefes, especialmente Dark Trials.

ATK básico + DMG de habilidade

  • O ATK básico não aumenta o dano do seu combo Rosa vs Vermelho não altera o valor, é o mesmo aumento para ambas as raridades, o que significa que tudo o que você ganhará é o buff de DMG de habilidade, que é bom para aumentar sua produção geral de DMG de habilidade, causando mais dano no início da batalha antes dos escudos serem ativados.

Agora que discutimos os principais, discutiremos rapidamente a classificação dos 3 últimos.

Eu diria que só obtê-los com o tempo e eles não têm pressa em adquiri-los, a razão é que Crit Res + Crit DMG + Counter DMG, todos esses valores são iguais, sejam eles rosa ou vermelhos, então você não está obtendo% adicional Vamos atualizá-lo para rosa, então tudo que você ganhará é Pal DMG, Skill Crit DMG e Pal Crit DMG.

Pal DMG é meio inútil para arqueiros e não é algo com que se preocupar, pois ajudará a causar pequenas quantidades de dano adicional, no entanto, não é uma alta prioridade, então a única outra coisa que está faltando é Skill Crit DMG, se você tiver alguma habilidade Crítico% em seus equipamentos para uma pequena mudança para ativar um Skill Crit, então valerá a pena atualizar este em vez dos outros, mas fora isso não é algo para se estressar ou se preocupar.

Agora, vamos discutir a atualização das almas. A prioridade mais alta seria Crit DMG e Combo DMG, esses 2 valores são muito importantes para aumentar sua produção de dano. Você também vai querer aumentar um pouco sua DEF e Crit res.

Digamos que neste cenário o meu nível de alma mais elevado seja 30, eu sugeriria algo semelhante a isto;

  • Combo DMG Nível 30
  • Dano Crítico Nível 30
  • HP Global Nível 30
  • Global ATK Nível 25 (Cerca de 60-70% da sua alma de nível mais alto)
  • Def Level 20 (Cerca de 60% do nível de sua alma mais elevada)
  • Crit Res Nível 20 (cerca de 60% do nível de sua alma mais elevada)
  • ATK DMG básico nível 1 (a menos que você tenha rosa), então talvez 50% -40% do nível de suas almas principais).
  • Counter DMG Nível 1 (a menos que você tenha rosa), então talvez 40% -30% do nível de suas almas principais).

Agora vamos discutir as subestatísticas das almas rosa para cada alma individual;

  • ATK Soul – Combo Global DMG
  • Crit DMG Soul – Combo Global DMG
  • Combo Soul – Combo Global DMG
  • Alma ATK Básica – Global Combo DMG
  • HP Soul – Habilidade DMG Res
  • Def Soul – Aerotransportado
  • Crit Res Soul – Habilidade DMG Res
  • Counter Soul – Ignorar Airborne

Eles são um conjunto específico de atributos que você pode realmente ter para cada alma individual, aqueles que são Skill DMG Res são principalmente baseados em res, o que significa que você pode ter Combo, Counter, pal dmg res ou até mesmo pal crit rate, o único atributo que vai provavelmente será a habilidade dmg res, já que cada classe tem habilidade dmg, então isso ajudará contra todas as classes.

Acessórios traseiros para Legend of Mushroom Classe de Arqueiro


Agora vamos passar para o acessório traseiro:

Quando você nivela suas penas, você deseja atualizá-las em 10s

Vou explicar o que quero dizer. Como você pode ver na imagem deixei 2 penas no nível 61 e me concentrei usando meus níveis na primeira.

Por que é isso?

Faço isso porque a cada 10 há um aumento enorme. O primeiro aumento começa no nível 11, depois no 21, 31, 41, 51, 61, 71 etc.

Então, primeiro levarei minha pena para o nível 71, depois atualizarei a segunda para 71, depois a terceira para 71, depois levarei minha primeira para 81, depois para 81, etc.

Basicamente;

71 61 61 ENTÃO 71 71 61 ENTÃO 71 71 71 ENTÃO

81 71 71 ENTÃO 81 81 71 ENTÃO 81 81 81 ETC.

Agora que discutimos as penas, vamos entrar na seção de penas/talentos dourados.

Começaremos discutindo os caminhos de cada talento individual;

Então, se você atualizar o caminho de talento da fúria como arqueiro, sim, você ainda receberá o bônus em estatísticas, etc. É melhor distribuir seus pontos entre todos os 3 caminhos de talento para um maior aumento em sua força, o custo de trabalhar em apenas 1 classe de talento é extremamente caro e só deve durar mais um ano a partir de agora. É muito caro, então distribua seus pontos entre as 3 árvores de talentos.

Vamos começar a discutir o caminho para atualizar a fúria, depois o tiro com arco e, finalmente, a feitiçaria.

Aqui no guerreiro desça pela rota da esquerda (HP)
Razão; Aqui você tem aumentos de HP, ATK e DEF (o mesmo para todos os 3 caminhos)
Você também tem a opção de aumentar sua % de Cura, Evasão e Ignorar Evasão

FIM DO JOGO: Sua recompensa por este caminho após completá-lo é uma velocidade de ataque + taxa crítica e aumento de ataque que ajudará muito em fases ou batalhas longas, como chefe de confronto, ladrão de lâmpadas e peixes grandes mal-humorados.

Agora vamos discutir o caminho do tiro com arco:

Aqui você quer focar principalmente no Combo DMG, este é o valor mais importante aqui. Quando você tiver atualizado totalmente isso, poderá aumentar sua Taxa de Cura + Quantidade de Cura.

Caminho de Feitiçaria:

No caminho de feitiçaria você tem ATK Básico como uma opção para aumentar sua produção de combo / crit dmg, então esta é uma boa escolha, assim como o Skill DMG, Skill DMG é bom para causar dano adicional ao seu oponente antes que ele tenha um escudo ativado.

Conclusão:

ATK, HP e DEF são baratos e devem ser maximizados, se você ainda estiver no início do jogo, basta subir cada um até o nível 10 por enquanto e voltar a ele. Seus principais atributos nos quais você deseja focar são # 1 Combo DMG, # 2 Basic ATK DMG # 3 Skill DMG.

Eu economizo pontos para meu COMBO DMG, atualize-o, com meus tokens extras eu irei usá-los no Basic ATK DMG / Skill DMG e então economizarei até que esteja próximo do mesmo nível do combo dmg.

A razão pela qual não me concentro apenas no COMBO DMG em si é porque é caro e estarei perdendo muito Skill DMG + Combo/Crit DMG adicional com o aumento da atualização Basic ATK.

Despertando para Legend of Mushroom Classe de Arqueiro

Agora que discutimos a maior parte do jogo, vamos entrar no Despertar e no que focar,

Então, primeiro eu gostaria de dizer que no início do jogo, se você é novo, provavelmente é melhor comprar 1 alma rosa primeiro (Combo + Boss DMG) na loja de arena entre servidores e depois focar no despertar completo.

O despertar permite que você entre no próximo estágio da sua classe, concede 100k de poder adicionais e um pequeno aumento nas estatísticas.

Iremos numerá-los desta forma (por favor, tenha em mente que minhas estatísticas estão em todos os lugares depois de mudar inúmeras vezes para testes).

  • Canto superior esquerdo = 1
  • Meio Esquerdo = 2
  • Inferior Esquerdo = 3
  • Canto superior direito = 4
  • Meio Direita = 5
  • Inferior Direito = 6
  • 1 = ATK Global – Baixa Prioridade
  • 2 = Defesa – Alta Prioridade
  • 3 = Res Crítica – Alta Prioridade
  • 4 = HP – Alta Prioridade
  • 5 = Dano Crítico – Maior Prioridade
  • 6 = Dano de Ataque Básico – Maior Prioridade


Raciocínio; Dano Crítico aumenta seu DMG Crítico geral, causando uma quantidade maior de dano, Dano de Ataque Básico aumenta sua produção de DMG Combo

Defesa, Crit Res + HP têm aproximadamente a mesma prioridade, já que o dano do arqueiro é preferido devido aos guerreiros serem contra-orientados, se você causar uma diferença de ataque esmagadora em comparação com os guerreiros, você apenas destruirá seu HP.

O ATK Global é bom porque aumenta sua produção geral de DMG, no entanto, você já está obtendo isso com Dano Crítico + Dano de Ataque Básico, então não faz uma grande diferença.

Eu atualizaria algo assim:

  • 1 = ATK Global – Nível 3 (Mantenha 30% do nível de maior prioridade)
  • 2 = Defesa – Nível 6 (Mantenha 60% do nível de maior prioridade)
  • 3 = Crit Res – Nível 6 (Mantenha 60% do nível de maior prioridade)
  • 4 = HP – Nível 6 (Mantenha 60% do nível de maior prioridade)
  • 5 = Dano Crítico – Nível 10 (atualização quando disponível)
  • 6 = Dano de Ataque Básico – Nível 10 (Atualização quando Disponível)

Relíquias para a classe Arqueiro

Relíquias de Boss para Archer PVE

O Livro da Imunidade e o Livro do Couro de Besta – você pode alternar e ver qual o ajuda mais. Se você tiver alto dano de Combo/Crit, então o livro de couro de fera é útil para matar o chefe antes que ele toque em você, pois aumenta o DMG causado em 20%.

O Livro de Imunidade é bom se você tiver HP alto, mas ataque baixo, isso o torna imune a danos por 3 segundos quando seu HP cair abaixo de 70%.

Pessoalmente, sempre preferi o livro Beasthide porque tenho alto dano por segundo (DPS).

Relíquias de Boss para Archer PVP

O livro Stonewrit Tome e o livro Flamke. Esses 2 livros dependem de onde você está no jogo. Se você estiver no início do jogo, o tomo Stonewrit provavelmente o ajudará mais do que o livro em chamas. Se você estiver atrasado no jogo, sugiro usar o livro de chamas para obter velocidade de ataque extra, isso serve tanto para PVP quanto para PVE (estágios)

Relíquias do Tesouro para a Classe Arqueiro

As relíquias da Caixa Mágica e do Cristal Metamórfico. Essas relíquias e dependendo de quais amigos você está usando, Magic Box é um bom jogo inicial ao usar o amigo Angel Deer (é um cervo branco, a raridade é vermelha e tem raridades mais baixas deste amigo)

Se você estiver no final do jogo (estágio final do infinito/revelação) usando o Octopus Pal, então é possível usar o cristal metamórfico, mas a Magic Box provavelmente o ajudará mais se você estiver usando

o amigo da caixa mágica pela resistência a danos. Se você estiver optando por uma construção DPS, poderá usar o Octopus com o cristal.

Relíquias de Máscara para Classe Arqueiro

As relíquias Bonded Mask e Arrow King Mask. Essas máscaras dependem principalmente da taxa de Combo/Crit que você tem em seu personagem. Eu pessoalmente sempre usei a Bonded Mask para obter uma taxa crítica mais alta. Se você tiver uma taxa crítica mais alta do que sua% de combo, poderá usar a máscara Arrow King.

Relíquias Fósseis para Classe Arqueiro


As relíquias fósseis Thundercaller Kite e Dragonweave Circlet. Esses fósseis são usados ​​​​principalmente para arqueiros, eu uso principalmente Thundercaller Kite porque o valor do dano aumenta quanto mais a relíquia é nivelada, testei Dragonweave Circlet em vários momentos diferentes durante minha experiência LOM, ele não se mostra tão eficaz quanto o Kite. Mas Dragonweave é uma relíquia alternativa para usar se você tiver o amigo Dragão ou Lagarto em uso.

Relíquias de Estátua para Classe Arqueiro


As relíquias da Estátua da Energia e da Estátua do Tempo. A estátua de energia no início do jogo é boa para usar, pois você ainda não terá a bomba de fumaça ou o deslumbramento desbloqueados. Ter o dano extra durante o escudo de cogumelos lhe dará uma vantagem extra durante o PVP/PVE. Se você estiver no meio/tardio do jogo, é recomendado usar o Time Statue para os efeitos prolongados da bomba de fumaça e das habilidades ofuscadas.

Colar Relíquias para Classe Arqueiro


As relíquias do Colar da Foice Carmesim, do Colar do Espírito e do Colar da Tempestade. Esses colares são dependentes:

Se você usar Clone Strike, use Storm Necklace. No entanto, se seu clone morrer imediatamente, use o Spirit Necklace, se seu clone não estiver causando nenhum dano antes que seus efeitos de blitz/desarme acabem, então é inútil tê-lo.

Se você estiver no início do jogo e não tiver o ataque blitz / ataque clone desbloqueado, você pode usar a Foice Carmesim. Sugiro usá-la se você tiver rajada selvagem (rajada selvagem é boa para estágios PVE para o DMG AOE)
